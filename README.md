# Spotify-Display
This is my project for hackclub stasis it is a spotify display which displays current track playing with 3 buttons first stops/plays the track second goes to previous track and third skips track
## Setup
-  Download [**Arduino IDE**](https://www.arduino.cc/en/software/)
-  Download all libraries

<details>
<summary>how to download all libraries</summary>

- Follow [this **tutorial** to download esp32 core library](https://randomnerdtutorials.com/installing-the-esp32-board-in-arduino-ide-windows-instructions/)
- **Download** [Spotify Esp32](https://github.com/FinianLandes/SpotifyEsp32) as **.zip**
